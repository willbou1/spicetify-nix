Appendix A. Spicetify-nix Options

Below are the options provided by spicetify-nix provided in no particular order.

programs.spicetify.enable

Whether to enable Spicetify a modified Spotify…

Type: boolean

Default: false

Example: true

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.enabledCustomApps

Custom apps to add to the spice. See https://spicetify.app/docs/development/custom-apps.

Type: list of (attribute set of anything)

Default: [ ]

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.enabledCustomApps.*.name

Name of the app. No spaces or special characters

Type: string

Default: ""

Example: "localFiles"

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.enabledCustomApps.*.src

Path to the folder containing the app code.

Type: path in the Nix store

Example:

''
  pkgs.fetchFromGitHub {
    owner = "hroland";
    repo = "spicetify-show-local-files";
    rev = "1bfd2fc80385b21ed6dd207b00a371065e53042e";
    hash = "sha256-neKR2WaZ1K10dZZ0nAKJJEHNS56o8vCpYpi+ZJYJ/gU=";
  }
''

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.enabledExtensions

A list of extensions. See https://spicetify.app/docs/advanced-usage/extensions/.

Type: list of (path in the Nix store or (attribute set of anything))

Default: [ ]

Example:

''
  [
    {
      src = (pkgs.fetchFromGitHub {
        owner = "Taeko-ar";
        repo = "spicetify-last-fm";
        rev = "d2f1d3c1e286d789ddfa002f162405782d822c55";
        hash = "sha256-/C4Y3zuSAEwhMXCRG2/4b5oWfGz/ij6wu0B+CpuJKXs=";
      }) + /src;
  
      name = "lastfm.js";
    }
  ]
''

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.enabledSnippets

Snippets to add to the spice. See https://github.com/spicetify/marketplace/blob/main/resources/snippets.json.

Type: list of string

Default: [ ]

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.alwaysEnableDevTools

Whether to enable chromium dev tools.

Type: boolean

Default: false

Example: true

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.colorScheme

Spicetify color scheme to use, given a specific programs.spicetify.theme. If using programs.spicetify.customColorScheme, leave this as default ("custom").

Type: string

Default: ""

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.createdPackages

A list of all generated packages containing the spiced spotify and extra packages from the current theme.

Type: list of package (read only)

Default:

[ config.programs.spicetify.spicedSpotify ] ++ config.programs.spicetify.theme.extraPkgs

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.customColorScheme

Custom scheme used to generate a corresponding color.ini. See https://spicetify.app/docs/development/themes.

Type: attribute set of string

Default: { }

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.dontInstall

Whether to enable outputting spiced spotify to config.programs.spicetify.spicedSpotify, but not installing it…

Type: boolean

Default: false

Example: true

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.experimentalFeatures

Whether to enable experimental features.

Type: null or boolean

Default: null

Example: true

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.extraCommands

Extra commands to be run during the setup of spicetify.

Type: strings concatenated with “\n”

Default: ""

Declared by:

<spicetify-nix/modules/common.nix>

The final spotify package after spicing.

Declared by:

<spicetify-nix/modules/common.nix>

<spicetify-nix/modules/common.nix>
programs.spicetify.theme.src

Path to folder containing the theme.

Type: path in the Nix store

Example:

''
  fetchFromGitHub {
    owner = "spicetify";
    repo = "spicetify-themes";
    rev = "02badb180c902f986a4ea4e4033e69fe8eec6a55";
    hash = "sha256-KD9VfHtlN0BIHC4inlooxw5XC4xlHNC5evASRqP7pUA=";
  }
  Or a relative path 
  
  ./myTheme
''

Declared by:

<spicetify-nix/modules/common.nix>
programs.spicetify.windowManagerPatch

Whether to enable preloading the spotifywm patch.

Note: this is only available on linux .

Type: boolean

Default: false

Example: true
